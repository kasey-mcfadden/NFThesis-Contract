# NFThesis-Contract
 NFT smart contract for NFThesis project. Check it out on [Etherscan](https://ropsten.etherscan.io/address/0x5dda91c9EF7AA5103C3F69f06cd46bfE1E2A77c6).

Used Metamask, Solidity, Hardhat, Ropsten, and Alchemy.
